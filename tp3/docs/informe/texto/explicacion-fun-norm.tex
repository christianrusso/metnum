\subsubsection{Resolver Cuadrados Mínimos usando Funciones Normales}
Se puede demostrar\footnote{No lo haremos aquí.} que encontrar el $x$ que minimiza la norma de $Ax-b$ equivalente a resolver el sistema $A^tAx=A^tb$. La ventaja es que requiere de menos conocimientos teóricos sobre matrices para resolver el sistema dado que consta sólo de multiplicaciones entre matrices y luego resolver un sistema lineal de ecuaciones en forma matricial.

Sin embargo, una desventaja muy grande sobre otros métodos para resolver el problema de Cuadrados Mínimos consta de que el número de condición de $A^tA$ es el doble que el número de condición de la matriz $A$. Esto significa que si la matriz $A$ es muy mal condicionada, la matriz $A^tA$ lo es aún más.
